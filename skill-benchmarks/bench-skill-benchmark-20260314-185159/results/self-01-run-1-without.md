# Benchmark Tasks for Docker Compose Skill

## Task 1: Build a Docker Compose File

**Prompt:** Create a docker-compose.yml for a web app with a database and cache.

**Rubric:**
- correctness: Does it work?
- completeness: Does it have all services?
- pattern_adherence: Is it well structured?
- edge_cases: Does it handle errors?

## Task 2: Add Logging

**Prompt:** Add centralized logging to a Docker Compose setup with ELK stack.

**Rubric:**
- correctness: Is the ELK configuration correct?
- completeness: Are all logging components included?
- pattern_adherence: Does it follow good practices?
- edge_cases: Does it handle log rotation?

## Task 3: Scale Services

**Prompt:** Configure a Docker Compose file to scale a web service to multiple replicas.

**Rubric:**
- correctness: Does scaling work?
- completeness: Is load balancing addressed?
- pattern_adherence: Is the approach standard?
- edge_cases: Does it handle state?
