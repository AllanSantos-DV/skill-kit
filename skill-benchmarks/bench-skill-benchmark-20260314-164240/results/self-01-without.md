# Benchmark Tasks for Docker Compose Skill

Here are 3 benchmark tasks to evaluate the Docker Compose best practices skill.

## Task 1: Build a Docker Compose Setup

**Prompt:** "Create a docker-compose.yml for a web application with a database."

**Rubric:**
- correctness: The docker-compose.yml should work and be syntactically valid
- completeness: Should include all necessary services
- pattern_adherence: Should follow good practices
- edge_cases: Should handle common issues

---

## Task 2: Multi-Container Application

**Prompt:** "Write a Docker Compose configuration for a microservices architecture with 3 services that communicate with each other."

**Rubric:**
- correctness: Services should be able to communicate
- completeness: All services should be defined properly
- pattern_adherence: Should use Docker Compose features correctly
- edge_cases: Should handle service failures

---

## Task 3: Docker Compose for Development

**Prompt:** "Set up a Docker Compose development environment for a Python Flask app with PostgreSQL and Redis."

**Rubric:**
- correctness: The configuration should work for local development
- completeness: All three services should be included
- pattern_adherence: Should be well-organized
- edge_cases: Should handle data persistence
