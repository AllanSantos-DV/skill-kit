# Neural Link Runtime

Runtime inteligente que intercepta e otimiza hooks dos agentes Copilot.

## Instalação

Este runtime é distribuído como parte do **Skill Kit**. Para usar:

1. Adicione o Skill Kit no Skill Manager for Copilot
2. O runtime será detectado e instalado automaticamente
