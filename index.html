Como Usar na Prática

Exemplos concretos de como interagir com agents, skills e hooks no dia a dia dentro do VS Code.

🎯
1. Usando Agents

Agents aparecem no dropdown de chat do Copilot no VS Code e determinam comportamento, ferramentas e permissões.

  • Selecione @orchestrator para tarefas complexas — ele avalia, divide e delega para especialistas.
  • Use @researcher quando precisar apenas entender ou investigar sem alterar código.
  • Escolha @implementor para execução direta e mudanças disciplinadas no repositório.
Dicas rápidas →
📚
2. Usando Skills

Skills contêm guias e automações que o agent pode invocar automaticamente quando detecta um pedido no domínio correspondente.

  • Peça para criar um hook — o agent pode detectar e acionar a hooks-creator para seguir o fluxo completo.
  • Peça para criar um agente — o agent pode acionar agent-creator que orienta a criação com frontmatter e testes.
  • Skills também podem ser invocadas explicitamente pelo nome quando quiser forçar um comportamento específico.
Como funciona →
🔧
3. Hooks em Ação

Hooks são guardrails que rodam automaticamente em lifecycle events — o usuário normalmente não precisa executar nada manualmente.

  • Editou um arquivo sem ler primeiro? O hook read-before-edit injeta um lembrete para revisar antes de alterar.
  • Tentou commitar sem Conventional Commit? O hook pre-commit-guard bloqueia ou solicita correção.
  • Alterou tipos/contratos? O hook contract-check alerta consumidores possivelmente impactados.

🚀 Começando

Duas formas de instalar — escolha a que funciona melhor para você.

Via extensão Recomendado

Instalação automática via Skill Manager

  1. Instale a extensão Skill Manager for Copilot no VS Code
  2. Abra a paleta de comandos (Ctrl+Shift+P) e execute Skill Manager: Add Repository
  3. Adicione a URL deste repositório
  4. Execute Skill Manager: Pull All para instalar agents, skills e hooks
⬇ Instalar Skill Manager →

Manual

Clone e copie os arquivos manualmente

  1. Clone este repositório
  2. Copie agents/ para ~/.copilot/agents/
  3. Copie skills/ para ~/.copilot/skills/
  4. Copie hooks/ para ~/.copilot/hooks/
  5. Recarregue o VS Code
