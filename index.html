🚀 Começando

Via extensão (recomendado)

  1. Instale a extensão Skill Manager for Copilot no VS Code
  2. Abra a paleta de comandos (Ctrl+Shift+P) e execute Skill Manager: Add Repository
  3. Adicione a URL deste repositório
  4. Execute Skill Manager: Pull All para instalar agents, skills e hooks

Manual

  1. Clone este repositório
  2. Copie o conteúdo de agents/ para ~/.copilot/agents/
  3. Copie o conteúdo de skills/ para ~/.copilot/skills/
  4. Copie o conteúdo de hooks/ para ~/.copilot/hooks/
  5. Recarregue o VS Code
